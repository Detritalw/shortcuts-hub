# 如何加入？
1. `如果你从未贡献过本仓库` 在 shortcuts 文件夹中新建一个以你 Github 用户名为名的文件夹,如 Detritalw  
2. 在刚刚创建的文件夹中新建一个以`你的项目的名称`为名的文件夹  
3. 在刚刚创建的项目文件夹中上传 `.shortcuts 文件`
